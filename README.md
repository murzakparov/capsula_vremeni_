# Наша капсула времени 💌

Цифровой подарок для пары на расстоянии: Бишкек ↔ Нидерланды.

## Что внутри

- `index.html` — **готовый файл**. Всё уже собрано внутрь (React + стили). Просто открой его в браузере или залей на любой хостинг (Vercel, Netlify, GitHub Pages).
- `src/App.jsx` — главный React-компонент (часы, тема, капсула).
- `src/messages.js` — массив сообщений.
- `src/theme.css` — адаптивная тема день/ночь (CSS-переменные).
- `src/tailwind-lite.css` — скомпилированный набор Tailwind-утилит (сборка без интернета).
- `build.js` — скрипт сборки: `node build.js` (нужны `react`, `react-dom`, `esbuild`).

## Как добавить свои сообщения

Самый простой способ — открыть `index.html` в текстовом редакторе,
найти блок `window.MESSAGES = [...]` в самом верху файла и добавить строки:

```js
{ text: "текст сообщения", date: "ДД.ММ.ГГГГ" },
```

Пересобирать ничего не нужно — приложение читает `window.MESSAGES` напрямую.

## Проверка тем

Добавь к адресу `?theme=day` или `?theme=night`, чтобы посмотреть обе темы,
не дожидаясь нужного времени суток. Без параметра тема выбирается
автоматически по времени в Нидерландах (день: 6:00–18:00).

## Telegram-уведомления об обнимашках (v4)

Сборка теперь создаёт две страницы:

- `index.html` — страница для Сабинчик: её обнимашки приходят уведомлением Суле.
- `index-for-me.html` — страница для Сули: его обнимашки приходят уведомлением Сабинчик.

Настройка — в блоке `window.TELEGRAM` в самом верху каждого html-файла:

1. `botToken` — токен бота от @BotFather.
2. `chatIdSulia` / `chatIdSabina` — числовые chat_id личных чатов с ботом.
   Каждый должен открыть бота и нажать «Start» (бот не может писать первым!),
   затем открыть `https://api.telegram.org/bot<ТОКЕН>/getUpdates` и найти
   `"chat":{"id":...}`.
3. `viewer` не менять — он различает страницы.

Быстрые нажатия подряд склеиваются в одно сообщение (отправка через 1,8 с после
последнего клика). Пока токен/chat_id не заполнены, уведомления просто не отправляются.
Счётчики страниц независимы: `hugCount` (её) и `hugCountSulia` (твоя) в localStorage.
